# Formato para el Libro del Proyecto

## La forma

- **Tipos de letra**: Arial o Times New Roman
- **Tamaño de letra y estilos**
  - _Títulos_: 16pt, negrita, centrado
  - _Subtítulos_: 14pt, negrita, izquierda
  - _Cuerpo del trabajo_: 12pt, normal, justificado
- **Papel**: A4
- **Márgenes**
  - _Superior_: 2.54cm
  - _Inferior_: 2.54cm
  - _Izquierda_: 3cm
  - _Derecha_: 3cm
- **Interlineado**: 1.5
- **Espaciado**
  - _Anterior_: 0
  - _Posterior_: 10
- **Números de páginas**: La parte _pre textual_ enumerar con letra romana minúscula, centrado abajo. La parte _textual_ y _post textual_ con número arábigo, centrado abajo.
- **Encabezado**: Sin ningún tipo de escritura
- **Pie de página**: Sin ningún tipo de escritura
- **Total de páginas**: Mínimo 50 hojas a partir de la introducción.
